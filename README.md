# Template de Backend python

## Objetivo

Facilitar a implementação de backend em python

## Passo a passo para executar o projeto

> <b></b>
```
1- Criar repositorio privado no github
2- Clonar repositorio
```
> <b></b>
```
3- cd <pasta do repositorio>
4- git clone https://github.com/WILSON-SONS-SERVICOS-MARITIMOS-LTDA/template-flask-backend.git && mv  template-flask-backend/* . && rm -rf template-flask-backend
```
> <b></b>
```
5- Alterar todas as 'tags' nome_do_projeto
6- Alterar as variaveis de banco no arquivo run_app.bat:
  [<<porta_do_projeto - ex:5000>>, <<user_do_banco>>, <<password_do_banco>> ,<<host_do_banco>>]
7- Atribuir valor para as constantes ['API_TITLE','API_DESCRIPTION'] no arquivo ../view/bp/root_blueprint.py
8- Atribuir valor para a constante: 'BLUE_PRINT_BASE_URL' no arquivo ../util/constantes.py
```
> <b></b>
```
9- Executar o arquivo install_windows.bat
```
> <b></b>
```
10- Executar o script run_app.bat
```
